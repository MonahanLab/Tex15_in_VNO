This is all the relevant code and inputs to generate the analysis as seen on https://www.biorxiv.org/content/10.64898/2025.12.03.690614v1

For the Basescope code, the required file to load into the code is BaseScope_Trasnlation_Key.csv is required

For the cFos code, the required file to load into the code is Cfos_Translation_Key -Translation_Key.csv

For the resident intruder analysis the required files to load into the code are
    Worked RI - Copy of Resident Intruder Melina.csv
    Worked RI - Copy of Resident Intruder Sanjana.csv

For the Bulk RNAseq, the required files to load into the code are 
    ENSEMBL_plus_Barnes_with_Pcdh_constant_gene_reference.txt (the ENSEMBLE IDs assigned to the genes)
    V1R_Subfamilies.csv (List of V1R organised into families)
    V2Rs groups by Family - V1 Sheet.csv (list of V2Rs organised by family and whether they are C1 or C2 type)
    VNO_Libraries_Sheet8.csv (This is the file structure we use to pull in the relevant bam and bw files for processing)
    Vmn1r_annotation.csv (cluster annotations for V1Rs)
    Vmn2r_annotation.csv (cluster annotations for V2Rs)

For the SingleCell Code, we have ommitted the necessary Hills Jr. et al data from this repository but the raw data can be found at GSE252365
